# Kaarya Connect - Mobile-first React + Tailwind

This project is a mobile-first React + Vite + Tailwind reproduction of KaaryaConnect (only mobile responsive as requested).

## Quick setup (locally)
1. Install dependencies:
   ```
   npm install
   ```
2. Start dev server:
   ```
   npm run dev
   ```
3. Build:
   ```
   npm run build
   ```

## Deploy to Vercel
